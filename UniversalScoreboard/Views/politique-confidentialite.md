# Politique de confidentialité

**Dernière mise à jour : 18 mars 2026**

## Introduction

PointBoard respecte votre vie privée. Cette politique explique quelles données nous collectons, comment nous les utilisons, et vos droits.

**En utilisant PointBoard, vous acceptez cette politique de confidentialité.**

## Données collectées

### Données stockées localement (sur votre appareil)
PointBoard stocke les informations suivantes **uniquement sur votre appareil** :
- Profils de joueurs (noms, avatars, statistiques)
- Historique de parties
- Préférences et réglages
- Achats effectués (via StoreKit)

**Ces données ne sont jamais transmises à nos serveurs.**

### Données collectées à des fins publicitaires
Pour afficher des publicités personnalisées, nos partenaires publicitaires peuvent collecter :
- **Identifiant publicitaire** (IDFA)
- **Données d'utilisation** (temps passé dans l'app, interactions)
- **Données techniques** (modèle d'appareil, version iOS, langue)
- **Localisation approximative** (pays, région)

**Partenaires publicitaires** :
- Google AdMob
- Autres réseaux publicitaires tiers

Ces partenaires utilisent des cookies et technologies similaires pour personnaliser les publicités.

## Utilisation des données

Les données collectées servent à :
- **Afficher des publicités personnalisées** (nous et nos partenaires)
- **Améliorer l'expérience utilisateur**
- **Analyser les performances de l'application**
- **Optimiser nos campagnes publicitaires**

**Nous nous réservons le droit d'utiliser ces données à des fins commerciales, marketing ou publicitaires.**

## Partage des données

Vos données peuvent être partagées avec :
- **Nos partenaires publicitaires** (Google AdMob, etc.)
- **Nos sous-traitants techniques** (hébergement, analytics)
- **Les autorités** si la loi l'exige

**Nous ne vendons pas vos données personnelles à des tiers.**

## Consentement

En utilisant PointBoard, **vous consentez expressément** à :
- La collecte de vos données selon cette politique
- L'utilisation de vos données par nous et nos partenaires publicitaires
- Le partage de vos données avec des tiers mentionnés ci-dessus

Vous pouvez retirer votre consentement en désinstallant l'application.

## Désactiver les publicités

Pour supprimer les publicités et limiter la collecte de données :
- Achetez le **Bundle Complet** dans l'application

Vous pouvez également limiter le suivi publicitaire dans les réglages iOS :
- **Réglages > Confidentialité > Suivi > Désactiver "Autoriser les demandes de suivi"**

## Conservation des données

- **Données locales** : conservées jusqu'à suppression de l'app
- **Données publicitaires** : conservées selon les politiques de nos partenaires (généralement 12-24 mois)

## Vos droits (RGPD)

Si vous résidez dans l'UE, vous disposez des droits suivants :
- **Droit d'accès** : connaître les données collectées
- **Droit de rectification** : corriger vos données
- **Droit à l'effacement** : supprimer vos données
- **Droit d'opposition** : refuser certains traitements

Pour exercer vos droits, contactez : **support@pointboard.app**

## Sécurité

Nous mettons en œuvre des mesures techniques raisonnables pour protéger vos données. Toutefois, **aucune méthode de transmission ou de stockage n'est 100% sécurisée**.

## Modifications

Cette politique peut être modifiée à tout moment. Les changements prendront effet dès publication dans l'application.

Nous vous encourageons à consulter régulièrement cette page.
